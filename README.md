# Taskify

A very specialised program specifically for copypasting
task descriptions from Moodle to the specified file.
Helpful when you don't have that much time for copying
all these descriptions manually.

## Usage

`taskify <link-to-tasks> <filepath>`

Make sure that you have taskify in your 'path' or you
are in the directory that contains the executable.

# Example

Example.cs:
```Csharp
namespace Taskify;

public class Example
{
}
```

```console
~/source-dir/
> taskify https://edu.mmcs.sfedu.ru/mod/assign/view.php?id=20397 .\Example.cs
```
