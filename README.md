# Taskify

A very specialised program for copypasting task descriptions 
from Moodle to the specified file. Helpful when you don't 
have much time for copying all the descriptions manually.

## Usage

`taskify <link-to-tasks> -f <filepath>`

Preview:
`taskify <link-to-tasks> -p`

Make sure that you have taskify in your 'path' or you
are in the directory that contains the executable.

# Example

Example.cs:
```Csharp
namespace Example;

public static class Program
{
    public static void Main()
    {
        Console.WriteLine("Hello, Taskify!");
    }
}
```

```console
~/source-dir/
> taskify https://edu.mmcs.sfedu.ru/mod/assign/view.php?id=20397 -f .\Example.cs
```
